\chapter*{Text Guided Object detection}
\addcontentsline{toc}{chapter}{Testing and Results}
\label{chap:testing-results}

The proposed Text-Guided Object Detection model was evaluated using both quantitative performance metrics and qualitative visual inspection. The evaluation aims to verify the model's capability to accurately localize target objects specified by natural language prompts while maintaining stable learning behavior during training.

The following sections present the training performance, evaluation metrics, dataset statistics, and qualitative detection examples.

%====================================================

\section{Training Performance}

Figure~\ref{fig:training} presents the overall training process. The training and validation losses decrease steadily throughout the epochs, indicating successful convergence without noticeable overfitting. The evaluation metrics also improve consistently as training progresses.

\section{Qualitative Detection Results}

The proposed Text-Guided Object Detection model receives two inputs: an image and a textual prompt describing the target object. Instead of detecting all objects within the scene, the model identifies only the object corresponding to the given text query and generates its bounding box.

Figure~\ref{fig:detection} demonstrates an example where the model successfully localizes the requested object while ignoring unrelated objects in the same image.

\begin{figure}[htbp]
\centering
\includegraphics[width=0.85\textwidth]{Images/example_detection.png}
\caption{Example of text-guided object detection.}
\label{fig:detection}
\end{figure}
