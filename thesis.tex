%% Abstract:
%%
\abstract{%
\hspace*{\parindent} Workflow automation has become increasingly important for reducing repetitive manual tasks, improving efficiency, and enabling seamless integration 
between digital services. However, many existing automation platforms are either overly complex or offer limited flexibility for customization and learning.

This project presents the design and implementation of a node-based workflow automation platform that enables users to create and execute automated workflows through 
interconnected processing nodes. The system supports both event-driven and scheduled execution while adopting a modular architecture that separates workflow definition, 
execution, and integration components, improving maintainability and extensibility.

The proposed platform is intended as a lightweight solution for educational purposes and practical experimentation. Its architecture also allows future 
expansion through integrations with artificial intelligence services and Internet of Things (IoT) devices without affecting the core system. 
By combining simplicity, extensibility, and workflow orchestration capabilities, the proposed system provides a practical foundation for learning 
and implementing workflow automation.

}
